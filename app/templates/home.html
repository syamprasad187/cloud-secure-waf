Cloud-Based Secure Web Application Firewall

Welcome to my project, a Cloud-Based Secure Web Application Firewall (WAF) built using AWS and Flask. This application is designed to protect web applications from malicious requests by leveraging machine learning and cloud infrastructure.

Project Overview

This WAF integrates a Flask web app with pages for home, registration, login, and about, hosted on AWS Elastic Beanstalk. It uses a Random Forest machine learning model to detect and block malicious HTTP requests, such as SQL injections or suspicious patterns, ensuring secure access for legitimate users.

Key Features

- Malicious Request Detection: Identifies threats using ML based on request features like path length, headers, and body content.
- IP Blocking: Temporarily blocks IPs sending malicious requests.
- AWS Free Tier: Deployed entirely within AWS Free Tier limits to avoid costs.
- Modern UI: Responsive design with a consistent navbar and card-based forms.

How It Works

When a request is made, the WAF middleware analyzes it. If deemed safe, the user accesses the page; if malicious, the request is blocked, and the IP is added to a block list. This ensures real-time protection without compromising performance.
